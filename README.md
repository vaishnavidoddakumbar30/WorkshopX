# WorkshopX 🚀

Futuristic workshop platform built with Next.js, MongoDB, and Tailwind.

## Features
- Registration system
- QR Event Pass + PDF download
- Admin dashboard UI
- Leaderboard
- Neon UI + animations

## Tech Stack
- Next.js (App Router)
- MongoDB (Mongoose)
- Tailwind CSS

## Setup
```bash
npm install
npm run dev
```
